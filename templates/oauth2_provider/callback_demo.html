What happens next in a real app:

1
Exchange code for token:

POST to /o/token/ with the authorization code, client credentials, and redirect URI

2
Get user data:

Use the access token to call /oauth/user/ to get the selected identity context data

3
Create/update account:

Use the identity data to create or update the user's account in your application

Sample Integration Code:

# Python example for token exchange
import requests

# Exchange authorization code for access token
token_response = requests.post('{{ request.scheme }}://{{ request.get_host }}/o/token/', data={
    'grant_type': 'authorization_code',
    'code': '{{ code }}',
    'redirect_uri': 'YOUR_REDIRECT_URI',
    'client_id': 'YOUR_CLIENT_ID',
    'client_secret': 'YOUR_CLIENT_SECRET',
})

access_token = token_response.json()['access_token']

# Get user data with selected context
user_response = requests.get('{{ request.scheme }}://{{ request.get_host }}/oauth/user/', headers={
    'Authorization': f'Bearer {access_token}'
})

user_data = user_response.json()
# user_data contains the identity info from the context the user selected
